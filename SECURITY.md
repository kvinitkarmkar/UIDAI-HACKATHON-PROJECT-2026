# Security Policy

Thank you for helping improve the security of this project.

## Supported Versions

Only the latest version of this repository receives security updates.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ Yes |
| Older   | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not disclose it publicly.

Please report it by opening a GitHub Security Advisory (if enabled) or by creating a private issue/discussion with the repository owner.

When reporting a vulnerability, please include:

- Description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Affected version (if known)
- Screenshots or proof of concept (optional)

Every valid report will be reviewed, and appropriate action will be taken.

Thank you for helping make this project more secure.
